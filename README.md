# Socioplastics

Socioplastics is a long-duration transdisciplinary research framework by Anto Lloveras, developed through LAPIEZA-LAB (Madrid) since 2009.

It operates across architecture, urban theory, epistemology, systems theory, conceptual art and media theory as a distributed epistemic infrastructure composed of indexed nodes, structured books, conceptual cores and machine-readable corpora.

## Current Status

- 3,000 indexed nodes  
- 30 books  
- 3 tomes  
- 6 conceptual cores  
- machine-readable corpus  
- DOI-anchored research objects
